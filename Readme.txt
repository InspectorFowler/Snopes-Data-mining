Fall CSCE Course Project on data mining involving web crawling and scraping using the rvest R package and performing textual analysis to create a predictive model for misinformation classification. The repository consits:-
1.  CSV files have been numbered in the order they were pulled from the website starting from the main links file to the sub category file, unprocessed whole data and finally the processed file with all the html codes removed.
2.  R codes separated on the basis of their functionality
